# FitBuddy Frontend

This folder contains the starter React frontend for the FitBuddy capstone project.

## Current state
- Vite + React app is running with JavaScript entry files.
- Material UI is installed and configured.
- React Router is set up with placeholder pages.
- A custom theme with the requested colors is ready.
