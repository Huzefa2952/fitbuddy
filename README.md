# FitBuddy

This repository contains the initial starter project structure for the FitBuddy capstone app.

## Structure
- Backend: Flask app with placeholder routes
- Frontend: React + Vite app with Material UI and React Router

## Run the frontend
1. Open the frontend folder.
2. Run `npm install` if needed.
3. Run `npm run dev`.

## Run the backend
1. Create and activate a Python virtual environment.
2. Install dependencies with `pip install -r backend/requirements.txt`.
3. Start the Flask app from the backend folder with `python app.py`.
